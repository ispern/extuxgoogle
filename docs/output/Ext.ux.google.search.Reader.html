Class Ext.ux.google.search.Reader

Package:Ext.ux.google.search
Defined In:Ext.ux.google.search.Reader.js
Class:Reader
Extends:JsonReader
Author:Kyte Project
Version:0.1

This class extends Ext.data.JsonReader and reads data passed from Ext.ux.google.search.Proxy and create an Array of Ext.data.Records.

Private method "getJsonAccessor" is overridden to accept "undefined" property which frequently happens in results from Google Search API
